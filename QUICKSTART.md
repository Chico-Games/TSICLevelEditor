# Quick Start Guide

Get started with the Biome Level Editor in 5 minutes!

## Step 1: Open the Editor

Simply open `index.html` in your web browser. No installation needed!

## Step 2: Select a Color

1. Look at the **Color Palette** in the left panel
2. Click on any color to select it (e.g., "Grassland" - green)
3. The selected color shows at the top with its name

## Step 3: Start Drawing

1. Make sure **Pencil** tool is selected (it's selected by default)
2. Click and drag on the canvas to paint tiles
3. Try different colors!

## Step 4: Try Other Tools

Click on different tools in the left panel:
- **Bucket** - Fill large areas quickly
- **Line** - Draw straight lines
- **Rectangle** - Draw rectangular regions
- **Eraser** - Remove tiles

## Step 5: Use Layers

1. Look at the **Layers** panel on the right
2. Click on "Structures" to make it active
3. Draw some structures
4. Click on "Terrain" to switch back
5. Toggle visibility with the checkboxes

## Step 6: Save Your Work

1. Click **Save** in the top toolbar
2. Your level downloads as a JSON file
3. To load it later, click **Load** and select the file

## Tips for Beginners

### Keyboard Shortcuts
- `B` - Pencil tool
- `G` - Bucket fill
- `E` - Eraser
- `Ctrl+Z` - Undo
- `+/-` - Zoom in/out

### Navigation
- **Mouse Wheel** - Zoom
- **Shift+Click+Drag** - Pan around
- **Fit Button** - Reset view to see entire grid

### Mistakes?
- Press `Ctrl+Z` to undo (up to 50 steps!)
- Use the **Eraser** tool to remove specific tiles
- Click **New** to start fresh (it will ask to confirm)

## Example Workflow

1. **Select Grassland** (green) and use **Bucket Fill** to create a base
2. **Select Ocean** (blue) and draw a river with **Pencil**
3. Switch to **Structures layer**
4. **Select Town** and place some settlements
5. Switch to **Decorations layer**
6. Add finishing touches
7. **Save** your masterpiece!
